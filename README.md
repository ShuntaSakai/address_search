# 郵便番号・住所検索ツール

郵便番号と住所を相互に検索できる、シンプルな Web アプリです。

## 概要

- フロントエンドは HTML / CSS / JavaScript のみ
- 検索 API は Netlify Functions 経由で呼び出し
- ブラウザから第三者 API を直接呼ばない構成
- JSONP は使用しない
- 入力中に候補を表示するオートコンプリート UI
- 公開時は Netlify にそのまま載せられる

## ファイル構成

address_search/

## 主な機能

- 郵便番号から住所を検索
- 住所から郵便番号を検索
- 入力中の候補表示
- 候補クリック / タップで確定
- キーボード操作対応
- 検索結果のコピー
- エラーメッセージ表示
- クリア機能
- モバイル / PC 両対応

## 検索モード

### 郵便番号から探す

- 郵便番号の途中入力で候補を表示します
- `1000001` と `100-0001` の両方に対応します
- 全角数字を半角に変換します
- 3 桁以上入力すると候補を取得します
- 結果表示は `100-0001`、コピーは `1000001` 形式です

### 住所から探す

- 住所の一部または全部で検索できます
- 2 文字以上入力すると候補を表示します
- 複数件ヒットした場合は候補一覧から選べます
- Enter キーでも候補を確定できます

## 候補表示

- 入力欄の直下に候補パネルを表示します
- 候補項目には郵便番号、住所、ふりがなを表示します
- `↑` `↓` で候補移動、`Enter` で確定、`Escape` で閉じます
- 候補取得中は `候補を探しています...` を表示します
- 候補は最大 10 件まで表示します

## 入力補助

### 郵便番号

- 前後空白の除去
- 全角数字の半角化
- ハイフン有無の吸収
- 表示時の `3桁-4桁` 整形

### 住所

- 前後空白の除去
- 全角スペースの通常スペース化
- 連続空白の整形

## バリデーション

### 郵便番号モード

- 未入力: `郵便番号を入力してください`
- 3 桁未満: 候補取得を行わず、補助メッセージのみ表示

### 住所モード

- 未入力: `住所を入力してください`
- 2 文字未満: 候補取得を行わず、補助メッセージのみ表示

## API エンドポイント

ブラウザ側が呼ぶパス:

- `/api/postal-search?postalCode=***`
- `/api/address-search?address=***`

Netlify 上では以下の Functions に転送されます。

- `netlify/functions/postal-search.js`
- `netlify/functions/address-search.js`

## セキュリティ方針

- JSONP は使用しない
- ブラウザから第三者 API を直接呼ばない
- 同一オリジンの `/api/*` のみ利用
- 候補描画を含め、外部データを HTML に埋め込む箇所では `escapeHtml()` でエスケープ
- `netlify.toml` の CSP とセキュリティヘッダーを維持
